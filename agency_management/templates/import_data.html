Command Line Import (Recommended)

Use this command to import your spreadsheet:

python manage.py import_spreadsheet /path/to/your/2025_Budget.xlsx G+

Steps:

  1. First, do a dry run: python manage.py import_spreadsheet /path/to/file.xlsx G+ --dry-run
  2. If the preview looks good, run the actual import: python manage.py import_spreadsheet /path/to/file.xlsx G+
  3. Check the dashboard and capacity views to see your imported data

What gets imported:

  • Revenue sheet: Creates clients and monthly revenue entries
  • Payroll sheet: Creates team members with roles and hourly rates
  • Projects: Creates general projects for each client
